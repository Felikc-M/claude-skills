# TRELY — Готовые рецепты страниц (live patterns)

Снято с боевых страниц trely.io и shipping.trely.io. Используй как точный референс для новых страниц того же типа.

---

## РЕЦЕПТ 1 — Главный лендинг (Home / trely.io)

### Header
- Логотип "Trely" слева (small, чёрный wordmark)
- Nav center: «Наши IT-сервисы», «Комьюнити», «Проверенные сервисы», «Услуги для партнеров»
- Right: **gold-pill кнопка «Telegram»** (`background: linear-gradient(to right, #C6A064, #A8854F); color: #fff; padding: 10px 24px; border-radius: pill;`)

### Section A — Hero (centered variant)
- **Centered layout** (не split)
- H1 центрирован в 2 строки: «Trely — твой мир / путешествий в одном месте»
- Sub центрирован, gray, 2 строки описания
- **Один CTA** dark по центру: «Отправить/Доставить»
- Под CTA — большая иллюстрация маскота (мальчик в шляпе "Bali" с пальмами по бокам)
- Иллюстрация на cream background с большой полупрозрачной аркой `#E8E2D2` за ней

### Section B — Сервисы Trely (Featured pairs)
- H2 центрирован
- **2 крупные карточки в ряд** (не 3-up grid):
  - **Trely Shipping** — `background: #273C3F; color: #fff; border-radius: 28px;` слева внутри: H3 white + параграф cream + dark CTA `Отправить/Доставить`; справа — painterly иллюстрация Shipping-героини с коробками
  - **Trely Car** — `background: #FFFFFF;` тёмный H3, описание серым, dark CTA `Поехать`; справа — иллюстрация жёлтой машинки с двумя девушками внутри
- Аспект карточек ~5:3
- Высота ~280-320px

### Section C — Русскоязычное комьюнити
- Split 60/40 на cream BG
- Слева: H2 «Русскоязычное комьюнити», параграф, **dark CTA «Telegram»** (закруглённый прямоугольник)
- Справа: **iPhone-моккап** с Telegram-чатом «TRELY | Bali» (показывает реальные сообщения сообщества)

### Section D — Проверенные сервисы (5-up grid)
- H2 центрирован, sub центрирован
- 5 карточек в одну строку (или 3 + 2): Обмен валют / Визы / Жильё / Аренда байков-машин / Поиск ресторанов
- Карточка: cream BG `#F4F1EA`, radius 20px, padding 24px, объект-иллюстрация справа, H4 + 1 строка описания слева, CTA «Перейти →» внизу gold-цвета

### Section E — Услуги для партнеров (3-up)
- H2 центрирован
- 3 карточки на cream BG (или на белом):
  - Каждая: gold ✓ в тёмно-зелёном круге сверху, H4, 2 строки описания
  - Темы: «Размещение рекламы», «Инфо-поддержка», «Партнёрские проекты»
- CTA внизу центр: **gold-pill «Связаться с нами»**

### Section F — FAQ
- H2 центрирован «Ответы на частые вопросы»
- Accordion, gold + индикатор, разделители светлые

### Footer
- Logo + копирайт слева
- **Large gold-gradient button «Telegram»** справа (как в header, но крупнее)
- Внизу строка тонким серым: «Trely, 2026 | Все права защищены» + «Политика конфиденциальности» + «Остались вопросы? Пишите — мы на связи 24/7»

## РЕЦЕПТ 2 — Продуктовая/маршрутная страница (Shipping route)

Пример: «Отправить посылку Москва — Нью-Йорк». Используй для всех маршрутных и продукт-страниц.

### Header
- Logo "Trely" слева
- Nav: «Отправить/Доставить», «Маршруты», «Объявления»
- Right: **gold-pill «Создать заказ»** (вместо «Telegram» как на главной)

### Section A — Hero (split variant)
- Grid 55/45
- **Слева:**
  - H1 (huge Bold): «Отправить посылку / Москва — Нью-Йорк / с попутчиком» (разрыв строк ручной для баланса)
  - Sub 2-3 строки серым
  - **Stat strip** — 4 метрики в ряд через разделитель:
    ```
    3-7 дней    │  от $40       │  7 500 км      │  Ежедневно
    срок        │  тип. стоимость│  расстояние    │  Рейсы
    ```
    Числа золотом, лейблы серым
  - 2 CTA: dark «Создать заявку отправителю» + outline «Посмотреть предложения попутчикам»
- **Справа:** painterly иллюстрация главного маскота (парень в зелёном свитере) с коробками и чемоданом, на полупрозрачной арке

### Section B — Как отправить (4-step horizontal)
- H2 «Как отправить посылку по маршруту [город] — [город]»
- 4 карточки в ряд: номер шага в круге gold, мини-скриншот шага, H4, описание
- Шаги: Создайте заявку → Найдите путешественника → Согласуйте передачу → Подтвердите доставку

### Section C — Что можно отправить (6-up image cards)
- H2 «Что можно отправить из [город] в [город]»
- 6 категорий: Документы / Электроника / Одежда / Книги / Аксессуары / Подарки
- Каждая карточка:
  - Cream BG `#F4F1EA`, radius 20px
  - Painterly иллюстрация объекта (книги, ноутбук, рубашка, очки и т.п.)
  - H4 название категории
  - Маленькая ссылка «Смотреть →» gold
- Grid 3×2 на desktop, 2×3 на tablet, 1×6 на mobile

### Section D — Особенности доставки (Two-col trust table)
- H2 «Особенности доставки посылок в [страна]» + sub
- 2 колонки рядом:
  - **Колонка 1:** «Таможня [страна]» — заголовок с эмодзи флага
  - **Колонка 2:** «Требования к упаковке» — заголовок с иконкой коробки
- Внутри каждой колонки: bullet list с золотыми ✓ и пунктами
- Carрточки прозрачные на cream BG, разделитель — тонкая линия gold между колонками
- Опционально: над каждым списком gold-pill категория

### Section E — Тёмно-зелёная route card
- **На тёмно-зелёном фоне `#273C3F`** — отдельная секция-карточка `border-radius: 32px;`
- H2 cream: «Маршрут доставки из [город] в [город]»
- Внутри:
  - **Route block:** «Москва [airports] → 7 500 км (с иконкой самолёта золотой) → Нью-Йорк [airports]»
  - **Stats row:** 4 цифры gold под маршрутом: «7 500 км», «10-12 ч», «Ежедневно», «Стамбул, Дубай (пересадки)»
- Cream/gray текст для описаний, gold для цифр и иконки самолёта

### Section F — Через какие аэропорты (Two-col airport list)
- H2 «Через какие аэропорты проходит доставка»
- 2 колонки: **Отправление** / **Прилёт**
- В каждой колонке: список аэропортов с **gold airport code справа** (SVO, DME, VKO / JFK, EWR, LGA)
- Карточки строки: cream, чёткие радиусы

### Section G — Почему через Trely (split: features + character)
- H2 «Почему отправлять посылки через Trely выгоднее»
- Layout: **слева 2×3 grid с фичами** (✓ + H4 + 1 строка), **справа большой painterly portrait главного маскота**
- 6 фич: «Дешевле курьерских служб», «Передача из рук в руки», «Широкая география», «Безопасная передача», «Гибкие маршруты», «Поддержка 24/7»

### Section H — Популярные направления (carousel)
- H2 «Популярные направления»
- Карусель из 4-5 карточек-городов:
  - **Photo** города сверху (НЕ painterly — реальное фото или стилизация)
  - Снизу — gold-pill с названием города (Бали, Таиланд, Париж, Кейптаун, …)
- Стрелки вправо для прокрутки

### Section I — FAQ + Footer
Как в Рецепте 1.

## РЕЦЕПТ 3 — Шаблон новой продуктовой страницы (Trely Car / Stay / Shop)

Используй структуру Рецепта 2 как базу, но адаптируй:

| Секция | Trely Car | Trely Stay | Trely Shop |
|---|---|---|---|
| Hero stat strip | дней / от $X / км / отправлений в день | город / от $X/ночь / гостей / опции | страна / минзаказ / срок / категорий |
| Section "Как" | Создать поездку → Найти попутчика → Поехать → Оплатить | Найти жильё → Связаться → Забронировать → Заехать | Создать заявку → Найти шопера → Получить → Оплатить |
| Trust table | Документы водителя / Правила поездки | Документы заселения / Правила дома | Лимиты товаров / Декларация |
| Route card | Маршрут + время + остановки | Карта района + расстояния до точек | Маршрут товара + статусы |
| Категории | Типы поездок (городская/межгород/тур) | Типы жилья (квартира/вилла/отель) | Категории товаров |

## РЕЦЕПТ 4 — Лендинг сервиса экосистемы (Trely Travel Club / Services)

Для сервисов **без транзакции** (сообщество, контент, подбор).

### Структура:
1. Hero centered + один CTA (Telegram / Войти)
2. **Что вы получаете** — 4-6 фич с gold ✓
3. **Внутри клуба** — карусель/grid карточек с городами/направлениями
4. **iPhone-моккап** Telegram чата с примерами активности (как в Section C Рецепта 1)
5. **Тарифы** — 3 карточки (Базовый $0 / Premier $25 / Elite по приглашению) — в стиле "слиток золота"
6. **Отзывы** — горизонтальная карусель карточек с круглыми портретами
7. FAQ + Footer

## Cross-cutting: что обязательно для ВСЕХ страниц

✓ Лого "Trely" в header, ссылка на главную
✓ В header справа — **gold-pill CTA**, отличающийся от dark primary в hero
✓ Hero c хотя бы одной painterly иллюстрацией маскота
✓ Move | Live | Connect — упомянуть хотя бы раз (footer, divider, slogan)
✓ FAQ обязательно (даже 3 вопроса)
✓ Footer с большой Telegram-кнопкой gold-gradient справа
✓ Cream background `#F7F6F4`, секции на white или dark green для контраста
✓ Все cards radius 20-32px, мягкая тень
✓ Mobile: stack секций, padding 24px
