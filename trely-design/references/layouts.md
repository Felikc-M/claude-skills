# TRELY — Макеты слайдов (Layouts A-F)

Шесть проверенных композиций из официального pitch deck. Выбирай нужную по типу контента.

---

## Макет A — Заглавный/обложечный слайд

**Используй для:** обложки презентации, hero блока лендинга, финального слайда.

```
┌─────────────────────────────────────────────────────────┐
│  [nav links] [О нас] [Продукт] [Бизнес] [Предложение]   │  ← navigation pills (или просто лого слева)
│                                                         │
│                       [🎁 cube logo]                    │  ← gold/dark cube
│                                                         │
│                       Trely                             │  ← H1 huge Inter Tight Bold 120-160px
│                                                         │
│      Инфраструктура жизни в движении —                  │  ← Subtitle 22-28px Regular
│      соединяющая людей, сервисы и доверие               │
│                                                         │
│                Move │ Live │ Connect                    │  ← brand-divider
│                                                         │
│  ✉ email                              [иллюстрация       │
│  📱 telegram                           главного маскота, │
│  📞 phone                              большая, по центру│
│                                        снизу]            │
└─────────────────────────────────────────────────────────┘
```

CSS hints:
- Background: `#F7F6F4`
- Заголовок "Trely" — `font-weight: 700; font-size: clamp(96px, 12vw, 160px); letter-spacing: -0.03em`
- Логотип-куб над заголовком, золотой, ~64px
- Иллюстрация маскота — 50-60% ширины слайда, по центру или справа
- Опционально: огромная полупрозрачная "T" или арка `#E8E2D2` opacity 30% за иллюстрацией

**Используй для:** "Глобальный рынок", "Юнит-экономика", "Что мы делаем" — любой слайд с 3 параллельными карточками.

- Подпись под цифрой: `color: #6B6F71; font-size: 14px`

## Макет G — Тёмный data-слайд (бонус)

**Используй для:** financial deep-dive, экосистемная карта, "Реальный рост" — когда нужна вес и серьёзность.

```
┌─────────────────────────────────────────────────────────┐  ← background #273C3F
│ [gold logo]  О нас [Продукт] [Бизнес] ...        20│28  │
│                                                         │
│ Реальные пользователи. Реальные сделки.                 │  ← H1 WHITE
│ Подтверждённый спрос до масштабирования                 │  ← Subtitle gray-cream
│                                                         │
│  ┌─[dark card]──┐  ┌─[dark card]──┐    ┌─[gold pill]─┐  │
│  │ 6 500+       │  │ $300 000     │    │ Рост идёт   │  │
│  │ польз./мес   │  │ реализовано  │    │ органически │  │
│  └──────────────┘  └──────────────┘    └─────────────┘  │
│                                                         │
│  📈 LINE CHART (gold line on dark)                      │
│  Рост пользователей и заказов                           │
└─────────────────────────────────────────────────────────┘
```

CSS hints:
- BG `#273C3F` (или ещё темнее `#1A2A2D`)
- Карточки `background: #1A2F2F; border-radius: 20px;` — на 1 шаг светлее BG
- Текст white/cream, цифры — gold
- Графики: золотая линия, тонкие cream сетки

## Универсальные элементы для ВСЕХ макетов

### Header (top-bar)
```html
<header class="slide-top">
  <div class="logo"><!-- маленький cube logo + "Trely" --></div>
  <nav class="nav-pills">
    <span>О нас</span>
    <span class="active">Продукт</span>     <!-- active = warm pill #EDE4D1 -->
    <span>Бизнес-модель</span>
    <span>Предложение</span>
  </nav>
  <div class="slide-counter">07 | 28</div>
</header>
```

### Footer
- Слева: section name (например "Логотип") серым 14px
- Справа: год "2026" серым 14px

### Минимальные паддинги слайда
- top: 56px (с header), bottom: 56-72px (с footer)
- left/right: 96-120px
