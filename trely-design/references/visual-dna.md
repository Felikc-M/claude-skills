## 2. Типографика

### Гарнитура
**Inter Tight** — компактный гротеск с высокой читаемостью. Кириллица обязательна.
- Google Fonts: https://fonts.google.com/specimen/Inter+Tight
- Веса: Light (300) · Regular (400) · Medium (500) · SemiBold (600) · Bold (700)

### Иерархия
| Уровень | Вес | Размер (desktop / slide) | Цвет | Применение |
|---|---|---|---|---|
| H1 (hero) | Bold | 72-96px | `#0F1014` | Главные заголовки слайдов и hero лендинга |
| H1 (slide) | Bold | 54-72px | `#0F1014` | Заголовок обычного слайда |
| H2 | SemiBold/Bold | 36-44px | `#0F1014` | Заголовки секций |
| H3 | SemiBold | 24-28px | `#0F1014` | Заголовки карточек |
| Subtitle | Regular | 20-24px | `#6B6F71` | Подзаголовок под H1 |
| Body | Regular | 16-18px | `#0F1014` | Основной текст |
| Caption | Regular | 12-14px | `#6B6F71` | Сноски, footer-text |
| Stat number | Bold | 56-96px | `#C6A064` | Главные цифры в карточках |
| Stat label | Regular | 14-16px | `#6B6F71` | Подпись под цифрой |

### Правила набора
- **line-height:** 1.0-1.1 для крупных заголовков, 1.4 для body
- **letter-spacing:** −0.01em (плотно), для всех caps **+0.04em**
- **выравнивание:** заголовки — по левому краю (центр только для hero-обложки)
- **разрыв строки:** в крупных заголовках разрывай вручную, чтобы строки были визуально сбалансированы

### Акцентное слово в заголовке
Выделяй золотом (color `#C6A064`), **никогда курсивом**.
Альтернатива: ключевое слово оборачивается в **inline-пилюлю** — `background: #EDE4D1`, `border-radius: 999px`, `padding: 4px 14px`, цвет текста `#273C3F`.

Пример: «Сделать жизнь [независимой] от места» — где [независимой] в пилюле.

## 3. Логотип

### Знак
Изометрический куб (3 видимые грани), внутри — силуэт человека с поднятыми руками (символ путешественника, доверия, открытости). Линейный (line-art), толщина линии однородная.

### Версии
| Фон | Знак |
|---|---|
| Белый/Beige | Чёрный `#0F1014` |
| Тёмно-зелёный | Золотой `#C6A064` |
| Чёрный | Золотой `#C6A064` |
| Золотой | Чёрный |

### Wordmark
"Trely" — Inter Tight Bold. Открытая "y" (без завитка). Размер высоты строчных = высоте знака.

### Использование в макетах
Всегда **маленький** в верхнем-левом углу слайда/секции. Размер ≈ 32-48px по высоте. На обложке/закрывающем слайде — крупно по центру.

### ❌ Не делай
- Не заменяй типографику (✗ Times, Comic Sans и т.п.)
- Не меняй пропорции знака и буквенной части
- Не используй неутверждённые цвета (✗ оранжевый, синий)
- Не сжимай, не искажай, не наклоняй

## 4. Иллюстрации (главный визуальный код)

### Стиль
**Painterly digital illustration**, watercolor + gouache texture, видимое зерно и мазки, тёплый свет, semi-realistic пропорции с лёгкой стилизацией лица. Никогда: flat-vector, 3D-render, фотореализм, иконки в стиле Material/iOS.

### Цвета внутри иллюстрации
- Основные: deep forest green `#273C3F`, warm gold `#C6A064`, cream `#F7F6F4`
- Допустимые акценты: denim/lavender blue (одежда), tan/каштановый (волосы, дерево), tropical green (пальмы), warm brown (коробки)
- Чёрный — для волос и глаз

### Персонажи бренда

**1. Главный маскот** — молодой парень ~25 лет, азиатские черты лица, тёмные растрёпанные волосы, добрая улыбка, зелёный свитер с золотистыми точками-узором, держит смартфон. Появляется на обложке, заключительном слайде, hero лендинга.

**2. Shipping-героиня** — молодая женщина ~25 лет, длинные тёмные/каштановые волнистые волосы, голубой/деним свитер + лавандовые/фиолетовые брюки, бежит с большой картонной коробкой со звёздами. Динамичная поза — символ доставки.

**3. Car-дуэт** — две девушки в крошечной круглой машинке (deep teal/navy с золотыми колёсами). Водитель — тёмные волосы, пассажир — светлые/каштановые длинные.

**4. Travel Club маскот** — мальчик-подросток в плетёной шляпе с тёмной лентой, цветастая тропическая рубашка в зелёно-золотых тонах, держит деревянную табличку "Bali".

**5. Stay/Services** — мальчик в шляпе или путешественник, иногда в круглой виньетке с пальмами/морем на фоне.

### Объекты (иконы-иллюстрации)
- **Коробка** — картонная зелёно-tan с золотыми звёздами/наклейками, мягкая тень
- **Самолёт** — компактный, deep green фюзеляж с золотыми деталями, лёгкие облака
- **Машинка** — крошечная круглая, deep green кузов, золотые колёса
- **Пальма** — тропическая, мягкий watercolor стиль, кремово-зелёные тона
- **Чемодан** — denim blue с золотыми застёжками
- **Здание/Stay** — стилизованный домик/отель

### Композиция с иллюстрацией в макете
- **На светлом BG** — иллюстрация в круглой виньетке (border-radius: 50% или контейнер с радиусом), за ней огромная полупрозрачная арка `#E8E2D2` opacity ~40%
- **В hero** — иллюстрация полноразмерная справа/по центру, "плавающие" мини-карточки продуктов вокруг
- **В карточке сервиса** — объект (коробка / машина / пальма) центрирован сверху карточки

## 5. Радиусы, тени, отступы

### Border-radius
| Размер | Применение |
|---|---|
| 8px | input, маленькие кнопки |
| 12-16px | средние кнопки, badge |
| 20-24px | стандартная карточка |
| 28-32px | hero-карточки, большие блоки |
| 999px (pill) | теги, статус-пилюли, активная навигация |
| 50% | круглые виньетки для иллюстраций, портреты команды |

### Тени
Очень мягкие, тёплые. Никогда жёсткие.
- Карточка на cream: `0 4px 24px rgba(39, 60, 63, 0.06)`
- Карточка hover: `0 8px 32px rgba(39, 60, 63, 0.10)`
- Кнопка primary: `0 2px 8px rgba(15, 16, 20, 0.10)`
- Глобально избегай теней более ~32px blur

### Spacing scale
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128

### Сетка
- Слайд: 1920×1080, padding-h 96px, padding-v 80px, gap 24-32px между блоками
- Сайт desktop: max-width 1280-1440px, padding 64-96px по бокам, 12-col grid с gap 24px
- Mobile: padding 24px, single column, gap 16-24px

## 6. Декоративные элементы

### Арка / большой круг
На светлых макетах за иллюстрацией — большая окружность `#E8E2D2` (cream subdued), часто обрезанная краем экрана. Создаёт глубину и тёплый фон.

### Пальмовый силуэт
На слайдах с метриками — бледно-кремовые пальмы по краям с opacity 20-30%, создают тропический подтекст экосистемы путешествий.

### Inline-пилюля
Внутри строки заголовка ключевое слово оборачивается в скруглённый прямоугольник `#EDE4D1`. Padding 4-6 px vertical, 12-18px horizontal. Цвет текста — тёмно-зелёный `#273C3F`.

### Разделитель Move | Live | Connect
Под лого/слоганом: три слова через `|` (vertical pipe), серый цвет, Regular, 16-18px, letter-spacing +0.04em.

## 7. Чек-лист "это в стиле TRELY?"

Перед сдачей макета проверь:

- [ ] Фон — `#F7F6F4` (cream) или `#273C3F` (deep green). Никаких чисто-белых или серых
- [ ] Только три цвета в UI: зелёный + золото + кремовый (+ near-black для текста)
- [ ] Шрифт — Inter Tight, нигде не Roboto/Arial/SF Pro
- [ ] Заголовок Bold с tracking −0.01em
- [ ] Хотя бы одно ключевое слово выделено золотом или в пилюле
- [ ] Логотип-куб в верхнем-левом углу (для слайдов и hero)
- [ ] Если есть человек/объект — это painterly watercolor иллюстрация, не stock и не вектор
- [ ] Карточки скруглены 20-28px с мягкой тёплой тенью
- [ ] Цифры/метрики — золотом и крупно
- [ ] Нет синего, красного, фиолетового в UI
- [ ] Минимум 1 декоративный элемент (арка / пальма / пилюля) в композиции
- [ ] Move | Live | Connect — где-то присутствует (для брендовых поверхностей)
