# ДНК Клиента - глубинный анализ целевой аудитории

## Что делает скилл:

Проводит комплексный 5-частный анализ целевой аудитории с глубоким погружением в психологию клиента. Результат выдается в формате Markdown.

### 5 частей анализа:
1. **Идентичность, ценности и мировоззрение ЦА** - демография, убеждения, идентичность
2. **Боли, страхи и триггеры** - проблемы, эмоции, болевые ситуации
3. **Барьеры, страхи и прошлый опыт** - препятствия, мифы, неудачные попытки
4. **Жизнь после решения проблемы** - трансформация, ожидания рынка, возражения
5. **Триггеры и механика воздействия** - точки принятия решения, форматы, офферы

### Алгоритм работы:
- Диалоговый стиль - ведет пользователя по процессу
- Последовательная выдача по частям с паузами
- Гибкость - углубление в любой пункт по запросу
- Максимальная детализация без сокращений

## Использование:
